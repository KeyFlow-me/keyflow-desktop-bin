# KeyFlow Desktop Releases 🚀

Welcome to the official distribution repository for **KeyFlow Desktop**.  
KeyFlow is a high-fidelity, developer-centric workspace designed for seamless document management and professional publishing.

## 📦 How to Download

This repository hosts only the compiled binaries. To download the latest version for your operating system, please visit the **[Releases](https://github.com/taehwankwon/keyflow-desktop-bin/releases)** page.

### Supported Platforms
- **macOS**: `.dmg` (Universal, Apple Silicon & Intel)
- **Windows**: `.msi`, `.exe` (x64)
- **Linux**: `.AppImage`, `.deb` (Upcoming)

---

## ✨ Features (v2026.03.29-alpha01)

- **Premium Themes**: Includes official KeyFlow branding (Light/Dark), Midnight (Navy), and Aura (Amber Sunset).
- **Mermaid Support**: Native rendering of Mermaid diagrams with full theme synchronization.
- **State Persistence**: Remembers your window size, sidebar width, and the last file you edited.
- **Tauri-Powered**: Lightweight, secure, and blazing fast performance.

## 🛠 Installation

### macOS
1. Download the `.dmg` file from the latest release.
2. Open the `.dmg` and drag **KeyFlow** to your `Applications` folder.
3. *Note: As this is an alpha release, you may need to right-click and select "Open" to bypass macOS security checks.*

### Windows
1. Download the `.msi` or `.exe` installer.
2. Run the installer and follow the on-screen instructions.
